# 🎮 Tic-Tac-Toe Console Engine

A structural, command-line execution of the classic Tic-Tac-Toe game written in Python. The application allows a human user to compete directly against a randomized computer opponent.

## 🚀 Features
- **Dynamic Grid Engine:** Generates custom multi-line text blocks representing the board status.
- **Robust Input Validation:** Utilizes `try-except` blocks to securely catch invalid coordinate entries or string input crashes.
- **Logical Win Conditions:** Evaluates continuous structural arrays across rows, columns, and matrix diagonals to announce instant win/tie outcomes.
